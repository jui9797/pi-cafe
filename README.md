# ☕ Pi Cafe

A modern and responsive cafe website built with **Next.js**, **React**, and **Tailwind CSS**. The project showcases a stylish coffee shop experience with beautiful UI sections, featured menu items, customer reviews, and interactive components.

---

## 🚀 Live Website

🌐 [https://pi-cafe.vercel.app/](https://pi-cafe.vercel.app/)

---

# 📌 Project Overview

Pi Cafe is a modern coffee shop website designed to provide users with a smooth and visually appealing browsing experience.

The website includes:

* Hero banner section
* Featured coffee & menu items
* Customer reviews/testimonials
* About section
* Responsive navigation bar
* Beautiful animations and transitions
* Mobile responsive design
* Optimized image handling using Next.js

The main goal of this project is to create a premium cafe-style UI with clean structure, reusable components, and smooth user experience.

# ⚙️ Installation & Setup

Follow these steps to run the project locally.

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/pi-cafe.git
```

## 2️⃣ Navigate to the Project Folder

```bash
cd pi-cafe
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Run the Development Server

```bash
npm run dev
```

Now open:

```bash
http://localhost:3000
```

---

# 🏗️ Build for Production

```bash
npm run build
```

To start production server:

```bash
npm start
```
